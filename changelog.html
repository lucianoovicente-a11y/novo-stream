Notas de Atualização

Atualização 4.1.23 - Codename: "Conexão Direta"

Esta grande atualização introduz um módulo completo e dedicado para o gerenciamento de clientes P2P por código. O sistema foi projetado para ser robusto, seguro e totalmente integrado ao painel, com permissões específicas para administradores e revendedores.

🚀 Novas Funcionalidades

  • Módulo Dedicado de Clientes P2P por CódigoDESTAQUE

    Implementado um sistema completo para criar e gerenciar usuários P2P que acessam o serviço através de um código numérico único e uma senha global. Inclui a criação de clientes, testes com duração personalizável e renovação de planos.

  • Permissões para Administradores e Revendedores

    O módulo P2P distingue administradores e revendedores. Cada revendedor tem acesso apenas aos seus próprios clientes e testes P2P, garantindo a segurança e a organização das contas.

  • Templates de Mensagem Personalizáveis

    Adicionada uma nova área nas configurações para editar o template de mensagem que é gerado ao criar ou renovar um cliente/teste P2P, com botões para copiar e enviar via WhatsApp.

Atualização 4.1.22 - Codename: "Agilidade"

Esta atualização foca em otimizar o fluxo de trabalho dos revendedores com a introdução do Teste Rápido, e solidifica a confiabilidade do sistema com uma reestruturação completa do módulo de EPG, garantindo dados precisos e sempre disponíveis para os seus clientes.

🚀 Novas Funcionalidades

  • Sistema de Teste Rápido IntegradoDESTAQUE

    Agora é possível gerar um teste completo para um novo cliente com um único clique, diretamente do cabeçalho do painel. O sistema cria o usuário, define a validade e exibe todos os dados de acesso em um novo modal, otimizado para copiar e enviar via WhatsApp.

✨ Melhorias

  • Reestruturação Completa do Sistema de EPGDESTAQUE

    O módulo de EPG foi totalmente reescrito. A importação de dados agora é mais robusta, salvando toda a programação no banco de dados local. A entrega do guia aos aplicativos (`xmltv.php`) é instantânea e não depende mais de servidores externos, garantindo um EPG 100% funcional e confiável.

Atualização 4.1.21 - Codename: "Autonomia"

Esta atualização tem como foco principal aprimorar a autonomia dos seus clientes, permitindo que eles renovem seus planos diretamente da área do cliente, de forma automática e segura.

🚀 Novas Funcionalidades

  • Renovação Automática por PagamentoDESTAQUE

    Implementado um sistema de renovação automática que, após o cliente realizar o pagamento pela área do cliente, atualiza a data de vencimento do plano de forma instantânea, garantindo a continuidade do serviço sem a sua intervenção manual.

Atualização 4.1.20 - Codename: "Comunidade"

Esta versão foca em aprimorar a interação e eficiência, introduzindo o novo sistema de Chatbot e a gestão de listas em massa.

🚀 Novas Funcionalidades

  • Chatbot para Suporte em Tempo RealDESTAQUE

    Implementação de um chatbot para suporte ao cliente, permitindo interações em tempo real e respostas automáticas para dúvidas comuns.

  • Exclusão de Listas em Massa

    Uma nova ferramenta para selecionar e excluir múltiplas listas de clientes de uma só vez, agilizando a limpeza do painel.
